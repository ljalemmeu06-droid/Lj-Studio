# Security Policy

## Supported release

The current supported release is the latest public release of Updated Studio Software.

## Reporting security issues

Please report security issues privately to the project maintainer before publishing details publicly.

## Installer security

The preferred long-term installer path is:

1. Public source repository.
2. Reproducible release build.
3. Code signing through a trusted open-source signing provider such as SignPath Foundation.

Do not bypass Windows security warnings. Use trusted signing.
