# Privacy Policy

Updated Studio Software is designed to run locally on the user's computer.

## Data stored locally

The app may store project data, account test data, comments, revisions, and audit logs in local files on the user's computer.

## Network use

The app may contact network services only when a user chooses a feature that requires it, such as:

- AI transcription through a configured API endpoint.
- Sharing or cloud-sync features configured by the user.

If no external service is configured, the app can run as a local tool through `http://127.0.0.1:4174/`.

## Personal data

Users should not enter sensitive personal data unless they understand where the configured service sends it.
